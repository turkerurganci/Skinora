# Instructions

Read and follow these files at the start of every session:
- `.claude/CONTEXT.md` — proje bağlamı ve dosya haritası
- `.claude/INSTRUCTIONS.md` — çalışma talimatları
- `.claude/GUARDRAILS.md` — sınırlar ve yasaklar
- `.claude/PROMPTS.md` — prompt kütüphanesi
- `.claude/memory/MEMORY.md` — proje hafızası (user, feedback, project, reference)

Memory files are the single source of truth. When creating or updating memories, write to `.claude/memory/` and commit to the repo.

Cloud sessions (claude.ai/code): Environment setup script is `scripts/cloud-setup.sh` — installs .NET 9.0 SDK and git hooks.
